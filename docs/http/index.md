# 📄 HTTP

超文本传输协议（HTTP）是一种用于传输 HTML
等超媒体文档的应用层协议。它是为 Web
浏览器和 Web 服务器之间的通信而设计的，但也可用于其他目的。HTTP
遵循经典的客户端-服务器模型，客户端打开连接以发出请求，然后等待收到响应。HTTP
是一种无状态协议，这意味着服务器不会在两个请求之间保留任何数据（状态）
